# Arrays

## Array ante enti?

Array ante oka data structure. Same type yokka multiple elements ni store chese collection idi. Single variable name lo chala values store chesi, index dwara access cheyochu.

## Key Concepts

- **Fixed Size**: Size creation time lo decide avthundi
- **Index-Based Access**: 0 nundi start ayye indices tho elements access chestham
- **Contiguous Memory**: Elements adjacent memory locations lo store avthayi
- **Same Type**: Anni elements okate data type undali

## Example

```
Array: [10, 20, 30, 40, 50]
Index:   0   1   2   3   4
```

## Enduku use chestaru?

- **Fast Access**: Index tho direct ga element access (O(1) time)
- **Memory Efficient**: Elements continuous ga store avthayi
- **Easy Iteration**: Loop dwara anni elements easy ga process cheyochu
- **Multiple Values**: Related values anni okate chota store

## Common Operations

1. **Declaration**: Array create cheyadam
2. **Access**: Index use chesi elements read/modify
3. **Traversal**: Anni elements dwara loop cheyadam
4. **Search**: Specific element find cheyadam
5. **Insert/Delete**: Elements add/remove cheyadam

## Time Complexity

- **Access**: O(1) - Direct index access
- **Search**: O(n) - Linear search
- **Insert**: O(n) - Shifting kavali aite
- **Delete**: O(n) - Shifting kavali aite
