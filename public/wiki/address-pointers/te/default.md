# Memory Addresses and Pointers

## Memory address ante enti?

Computer memory lo data store ayye location ni identify cheyadaniki use chese unique identifier adi **memory address**.

## Pointer ante enti?

Memory address ni store chese variable ni **pointer** antaru.

## Key Concepts

- **Memory Address**: Data store ayina location
- **Reference**: Data undi place
- **Value**: Actual data
