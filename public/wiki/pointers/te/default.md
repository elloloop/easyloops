# Pointers

## Pointer ante enti?

Inka variable yokka memory address ni store chese variable ni **pointer** antaru.

## Key Concepts

- **Direct Memory Access**: Directly memory ni access cheyadam
- **Dynamic Memory**: Runtime lo memory allocate cheyadam
- **Data Structures**: Linked lists, trees create cheyadam

## Enduku use chestaru?

- Efficient data handling
- Memory control
- Dynamic data structures build cheyadam
- Functions lo data modify cheyadam

## Safety Issues

- Null pointer dereferencing
- Dangling pointers
- Memory leaks
- Buffer overflows
